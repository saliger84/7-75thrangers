# 7-75th Ranger Regiment — Static Website Template

This is a static HTML/CSS template designed to closely match the general layout/feel of 3rdinf.us:
- Sticky top navigation with dropdowns
- Left sidebar "panels" for navigation/sections
- Main content area with hero + cards + lists
- Multiple pages (Home, About, Calendar, Roster, Ranks, Awards, Training, News, Enlist)

## Run locally
Open `index.html` in your browser.

## Deploy free on GitHub Pages
1. Create a GitHub repo named: `<yourusername>.github.io`
2. Upload all files in this folder
3. Repo Settings → Pages → Deploy from Branch → `main` / root
4. Your site will be: `https://<yourusername>.github.io`

## Custom domain (optional)
Add a CNAME in GitHub Pages settings, then point your DNS to GitHub Pages.

Replace links, images, and roster/news content as needed.
