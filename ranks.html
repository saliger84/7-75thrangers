Rank Structure

Promotion requirements and rank information.

Enlist Today Learn More RanksPromotionLeadership
Colonel
O6
O6 | COL

Description

The Colonel commands the regiment and is the final authority on major decisions, standards, staffing, and operational direction.

Prerequisites

  • Appointed as Regimental Commander.
Lieutenant Colonel
O5
O5 | LTC

Description

Lieutenant Colonels serve as battalion-level commanders or executive officers and help drive planning, resourcing, and overall readiness.

Prerequisites

  • Six months time in grade as Major.
  • Appointed to battalion command or executive leadership.
Major
O4
O4 | MAJ

Description

Majors are field grade officers trusted with staff responsibilities, planning support, and executive leadership roles across the unit.

Prerequisites

  • Demonstrated excellence as a commissioned officer.
  • Promotion approval by command staff.
Captain
O3
O3 | CPT

Description

Captains command companies, manage operations, and develop junior leaders while enforcing standards across their formation.

Prerequisites

  • Successful platoon-level leadership.
Command Sergeant Major
E9
E9 | CSM

Description

The Command Sergeant Major advises the commander, enforces standards, and oversees enlisted development across the regiment.

Prerequisites

  • Senior NCO experience and command approval.
First Sergeant
E8
E8 | 1SG

Description

First Sergeants are the senior enlisted leaders of company formations and are responsible for discipline, mentorship, and readiness.

Prerequisites

  • Proven platoon-level leadership.
Sergeant First Class
E7
E7 | SFC

Description

Sergeant First Class personnel serve as platoon sergeants, senior staff NCOs, and section leaders.

Prerequisites

  • Leadership course completion.
Staff Sergeant
E6
E6 | SSG

Description

Staff Sergeants lead squads, run training, and mentor junior enlisted Rangers.

Prerequisites

  • Consistent squad-level leadership.
Sergeant
E5
E5 | SGT

Description

Sergeants lead fire teams and assist with training execution and discipline at the lowest leadership level.

Prerequisites

  • Recommended by chain of command.
Corporal / Specialist
E4
E4 | CPL / SPC

Description

Corporals and Specialists form the backbone of the unit and support team-level execution.

Prerequisites

  • Demonstrated reliability and performance.
Private First Class / Private
E3-E1
E3-E1 | PFC / PV2 / PVT

Description

Junior enlisted Rangers develop their core skills, complete training, and integrate into the unit structure.

Prerequisites

  • Successful enlistment and onboarding.
